Detailed Description

provide a relatively care-free version of the basic PE analogue operations

Introduction

Most of these macro provide a relatively carefree version of the PE operations. That is, they produce less error and handle the analogue polarity issue. For example, "A = pe.copy(B)" execute an idealized copy operation, with polarity of the image preserved and the offset error reduced.

Note, in these macros, the result AREG cannot be same as any of the source AREGs. For example, "A = pe.add(A,B)" is invalid.
